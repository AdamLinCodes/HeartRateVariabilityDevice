# Installation Instructions
1. Open termnal and use ```sudo apt-get install libqt5charts5 libqt5charts5-dev``` to install the QtCharts library, and its tools. **NB.** This needs to be done before step 2.
2. Use ```git clone https://github.com/AdamLinCodes/HeartRateVariabilityDevice.git``` to clone the repository. This will generate the main QT project folder to whichever directory your in.
3. Open QTCreator -> Open Project -> HeartRateVariabilityDevice.
4. In the bottom left corner, click the green run button.
